# CommitScope вЂ“ Implementation Plan

## Overview
Production-ready GitHub analytics SPA deployed on GitHub Pages. React 19 + Vite + TypeScript + Tailwind + shadcn/ui + Recharts + Framer Motion. GraphQL-first API strategy with aggressive localStorage caching.

---

## v1 Feature Scope (Core)

**Included:**
1. Landing page (hero, search, dark theme)
2. Profile card
3. Contribution overview (animated stat cards)
4. Contribution heatmap (GitHub-style calendar)
5. Contribution timeline chart (line chart with time range filters)
6. Repository explorer (cards, search, sort, filter)
7. Language analytics (pie chart + stats)
8. Developer journey timeline (animated vertical timeline)
9. Theme switching (dark/light/system)
10. Shareable URLs (`?user=octocat`)
11. Responsive design (mobile/tablet/desktop)
12. Skeleton loaders + empty/error states
13. GitHub Pages deployment via GitHub Actions

**Deferred to v2:**
- Before vs Now comparison
- Year comparison bar chart
- Monthly activity chart
- Coding patterns / heatmaps
- AI insights
- Achievements / badges
- Compare users
- Export (PNG/SVG/PDF)
- Repository timeline

## Architecture Decisions

### Routing
- React Router v7 with `HashRouter` for GitHub Pages compatibility (no 404 on refresh)
- Two routes: `/` (landing) and `/:username` (dashboard) вЂ” also parse `?user=` query param

### API Layer
- **GraphQL** (`api.github.com/graphql`): contribution data (commit calendar, PR counts, issue counts). Single query for user profile + contributions.
- **REST** (`api.github.com`): repository list, language stats. Simple endpoints, well-typed.
- **Cache**: localStorage with TTL (1 hour default). Key: `commitscope:cache:<endpoint>:<params>`. On cache hit, return stale data immediately, then revalidate in background.

### State Management
- TanStack Query for all API calls (built-in cache, stale-while-revalidate, retry)
- React Context for theme only
- URL state for selected user (`?user=`)

### Theming
- Tailwind `dark:` variant + CSS variables via shadcn/ui
- `useTheme` hook reads/writes localStorage + toggles `.dark` class on `<html>`
- Default: dark mode

### Animations
- Framer Motion for page transitions, card entrances, counter animations
- CSS transitions for hover effects (cheaper than JS)
- `AnimatePresence` for route changes

## Key Technical Details

### GitHub GraphQL Contribution Query
```graphql
query($login: String!) {
  user(login: $login) {
    contributionsCollection {
      contributionCalendar {
        weeks {
          contributionDays {
            contributionCount
            date
            color
          }
        }
        totalContributions
      }
      totalCommitContributions
      totalIssueContributions
      totalPullRequestContributions
      totalPullRequestReviewContributions
      restrictedContributionsCount
    }
    repositories(first: 100, orderBy: {field: STARGAZERS, direction: DESC}) {
      nodes { name, description, stargazerCount, forkCount, primaryLanguage { name, color }, updatedAt, url, isPrivate }
    }
  }
}
```

### Cache Strategy
```
Key: commitscope:cache:<type>:<username>:<params>
TTL: 1 hour (configurable)
On hit: return cached, revalidate in background
On miss: fetch, store, return
Stale indicator: show "Updated X minutes ago" badge
```

### Heatmap Color Scale (Dark Mode)
```
0 contributions: #161b22
1-3: #0e4429
4-6: #006d32
7-9: #26a641
10+: #39d353
```
